# Minimal Header Notes

## Recommended order

- `minimal-logo-glow`
  - Strongest default account banner.
  - Cleanest balance between presence and restraint.

- `minimal-logo-clean`
  - Most neutral option.
  - Best if you want the header to almost disappear.

- `minimal-logo-grid`
  - Best if you want a slightly more analytical/data-inflected background.

## Guidance

If the profile posts already carry the explanation, prefer `minimal-logo-glow`.
Use `minimal-logo-clean` if you want the quietest possible brand backdrop.
